# py-trading-tool

Simple trading tool I created to help me make trading decisions

## Dependencies
- python>=3.0
- homebrew https://brew.sh/
- pip `pip install -upgrade pip`

## Packages

#### Finance
````
 pip install yfinance
````
 
#### Tech analysis
````
 brew install ta-lib
 pip install ta-lib
````

#### General tools
````
 pip install matplotlib
 pip install numpy
 pip install pandas
````
